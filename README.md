# COVID-19-Chatbot
### 기간
- 2일 20.07.16 ~ 20.07.17
### 챗봇 아이디어 소개
- 기획 의도
  - 모든 확진자의 정확한 동선을 파악하기 어렵고 중복되는 경우가 많아 동선을 확인하는건 무의미한 시점이 됨
  - 자치구별 코로나19 확진자 현황을 크롤링하여 알림
- 기존 챗봇들의 문제점 : 너무 많은 정보를 제공하여 복잡
- 이점 : 원하는 정보만 간단하게 얻음
### 개발 환경
- Python, Flask, Dialogflow, ngrok, 라인 메신저, PyCharm
